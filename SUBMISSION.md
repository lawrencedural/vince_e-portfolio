### 2. Reflections for Week 1 and 2

#### Week 1 & 2: Initial Communication Challenges and Growth

**Title:** Overcoming Shyness and Building Confidence

**Content:**

At the beginning of this term, I struggled significantly with communication. I was too shy to approach my classmates and engage in small interactions. The thought of speaking in front of others or participating in group discussions made me extremely nervous.

**Key Challenges:**
- Difficulty expressing thoughts clearly during group work
- Anxiety when speaking in public
- Limited interaction with classmates due to shyness
- Fear of making mistakes while communicating

**Initial Reflections:**
During the first two weeks, I realized that my shyness was holding me back from fully participating in class activities. I noticed that other students seemed more confident and comfortable sharing their ideas. This made me reflect on why I was so anxious and what I could do to improve.

**Early Turning Points:**
- Participated in a one-minute silent communication exercise (eye contact and posture only)
- Delivered my first impromptu speech, which was challenging but eye-opening
- Started noticing that my classmates were supportive and non-judgmental

**Lessons Learned:**
- Communication anxiety is normal and can be overcome with practice
- Small steps lead to big changes
- My classmates were more accepting than I initially thought
- Confidence comes from taking action, not waiting for it to appear

**Growth Indicators:**
- Began volunteering for activities instead of waiting to be called on
- Started asking questions in class
- Felt slightly more comfortable in group settings

## How to Run

### Prerequisites:
- Node.js (v16 or higher)
- npm or yarn

### Installation:
```bash
npm install
```

### Development:
```bash
npm run dev
```

### Build for Production:
```bash
npm run build
```

### Preview Production Build:
```bash
npm run preview
```

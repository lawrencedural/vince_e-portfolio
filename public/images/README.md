# Images Folder

This folder contains all images used in the e-portfolio.

## Folder Structure:
```
images/
├── hero/          # Hero section headshot
├── about/         # About section portrait
├── projects/      # Project thumbnails
└── gallery/       # Thank you section gallery images
```

## Image Guidelines:

### General Recommendations:
- Use high-quality, professional images
- Optimize file sizes for web (use tools like TinyPNG or ImageOptim)
- Maintain consistent styling across all images
- Follow the specifications in each subfolder's README.md

### Image Formats:
- **JPG**: Best for photographs
- **PNG**: Best for graphics, logos, or images with transparency

### Naming Convention:
- Use lowercase letters
- Use hyphens (-) or underscores (_) instead of spaces
- Be descriptive (e.g., `hero-headshot.jpg`, `lesson-1-project.jpg`)

### File Size:
- Keep individual images under 500KB when possible
- For hero/about images, up to 1MB is acceptable if quality is maintained

## Updating Image Paths:

After adding images, update the paths in `src/data/config.ts` to match your actual filenames.

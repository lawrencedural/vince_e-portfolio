# Hobbies Images

Place your hobby images in this directory with the following naming convention:

## Required Images

1. **golf.jpg** - Single image for Golf hobby
2. **basketball.jpg** - Single image for Basketball hobby
3. **automotive-1.jpg** - First image for Automotive hobby
4. **automotive-2.jpg** - Second image for Automotive hobby

## Image Guidelines

- Recommended image format: JPG or PNG
- Recommended dimensions: 800x600px or larger (will be responsive)
- Keep file sizes optimized for web (under 500KB each)
- Ensure images are clear and well-lit

## Example Structure

```
public/images/hobbies/
├── golf.jpg
├── basketball.jpg
├── automotive-1.jpg
├── automotive-2.jpg
└── README.md
```

Once you add the images, they will automatically appear in the Hobbies page gallery.

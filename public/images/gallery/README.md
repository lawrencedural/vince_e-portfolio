# Gallery Images

Place photos for the "Thank You" section gallery here.

## Recommended Specifications:
- **Filenames**: `gallery-1.jpg`, `gallery-2.jpg`, `gallery-3.jpg`, etc.
  - Or update the paths in `src/data/config.ts`
- **Aspect Ratio**: 1:1 (square) - works best for gallery grid
- **Size**: 600x600px minimum (or higher resolution)
- **Format**: JPG or PNG
- **Style**: Professional photos, team photos, event photos, or meaningful images
- **Quantity**: 6 images (or update the array in `src/data/config.ts`)
- **Quality**: High resolution for crisp display

These images will be displayed in a gallery grid in the Thank You section. They will appear in grayscale and show color on hover.

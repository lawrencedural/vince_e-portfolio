# Hero Section Images

Place your professional headshot image here for the hero section.

## Recommended Specifications:
- **Filename**: `hero-headshot.jpg` (or update the path in `src/data/config.ts`)
- **Aspect Ratio**: 3:4 (portrait orientation)
- **Size**: 800x1067px minimum (or higher resolution)
- **Format**: JPG or PNG
- **Style**: Professional corporate headshot with neutral/soft gray background
- **Quality**: High resolution for crisp display

The image should be a professional headshot that represents you in a corporate/business setting.

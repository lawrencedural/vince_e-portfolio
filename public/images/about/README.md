# About Section Images

Place your professional portrait image here for the about section.

## Recommended Specifications:
- **Filename**: `about-portrait.jpg` (or update the path in `src/data/config.ts`)
- **Aspect Ratio**: 3:4 (portrait orientation)
- **Size**: 600x800px minimum (or higher resolution)
- **Format**: JPG or PNG
- **Style**: Professional portrait, can be same as hero or different angle
- **Quality**: High resolution for crisp display

This image will be displayed alongside your bio in the About section.

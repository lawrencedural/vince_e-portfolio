# Professional Documents

Place your downloadable professional documents here.

## Required Files:

### 1. Resume
- **Filename**: `Vince_Umali_Resume.pdf` (or update the path in `src/data/config.ts`)
- **Description**: Your professional resume highlighting education, experience, and skills

### 2. CV (Curriculum Vitae)
- **Filename**: `Vince_Umali_CV.pdf` (or update the path in `src/data/config.ts`)
- **Description**: Comprehensive CV with detailed academic and professional background
- **Note**: If you don't have a separate CV, you can use the same file as your resume

### 3. Cover Letter
- **Filename**: `Cover-Letter-Umali.jpg` (or update the path in `src/data/config.ts`)
- **Description**: A template or sample cover letter

## File Format:
- **Format**: PDF (recommended for professional documents)
- **Naming**: Use consistent naming with underscores or hyphens (no spaces)
- **Size**: Keep files optimized (under 5MB recommended)

## Important:
After adding your PDF files, make sure to:
1. Update the file paths in `src/data/config.ts` if you use different filenames
2. Test the download links to ensure they work correctly
